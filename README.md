## 🛠 How I Work — Vibe Coding Methodology

1. **Define requirements in natural language** (Korean/French) → design the architecture in conversation with Claude Code
2. **Version-control prompts like code** — system prompts live in `docs/system-prompts/` with versions, changelogs, and rollback procedures
3. **Multi-model orchestration** — the right model for each job: Claude (reasoning & structured output), Gemini (image & scene planning), Kie.ai/Seedance (video generation)
4. **Human directs and verifies, AI implements** — the commit history is the record of that process
